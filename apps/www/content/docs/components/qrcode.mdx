## Usage

The QR code is built using `qr-code-styling` and is pre-configured to automatically sync with your shadcn/ui theme variables.

```tsx
import QRCode from "@/components/ui/qrcode"

export default function Example() {
  return (
    <QRCode 
      value="https://deltacomponents.dev" 
      size={200} 
      dotStyle="rounded"
    />
  )
}
```

### Clickable Navigation

Enable the `enableNavigation` prop to transform the QR code into a functional link, allowing desktop users to click the code to visit the destination.


```tsx
<QRCode value="https://example.com" enableNavigation={true} size={200} />
```
## Notes

### Theme Awareness

Unlike static QR generators, this component listens for changes to the `dark` class on the `<html>` element.  
It uses a helper to resolve CSS variables like `--color-background`, ensuring the QR code remains readable
immediately after a theme toggle without requiring a page refresh.

### Scannability & Error Correction

When adding a center logo or using complex dot styles, scannability can decrease.

- Use `level="H"` (High) or `level="Q"` (Quartile) for error correction when a logo is present.
- Ensure a high contrast ratio between `bgColor` and `fgColor`.
- Avoid using `logoSize` greater than `0.3`, as it may obscure too much data for the reader to recover.

### SVG vs Canvas

The component renders as an SVG by default. This ensures the QR code remains crisp at any scale and is
easier to style via CSS if needed, though most styling should be handled via props for cross-browser
consistency.


## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | — | Required. The data or URL to encode. |
| `size` | `number` | `200` | The width and height in pixels. |
| `bgColor` | `string` | `"--color-background"` | Supports CSS variables or hex/hsl colors. |
| `fgColor` | `string` | `"--color-foreground"` | Supports CSS variables or hex/hsl colors. |
| `level` | `"L" \| "M" \| "Q" \| "H"` | `"M"` | Error correction level (higher allows more data recovery). |
| `dotStyle` | `enum` | `"square"` | Options: `rounded`, `dots`, `classy`, `classy-rounded`, `square`, `extra-rounded`. |
| `logoImage` | `string` | — | URL or Data URI of the center logo. |
| `logoSize` | `number` | `0.2` | Scale of logo relative to QR size (0 to 1). |
| `enableNavigation` | `boolean` | `false` | If true, clicking the component opens the value in a new tab. |
| `marginSize` | `number` | `0` | Quiet zone margin around the code. |
