## May 2026 - Initial Release 🎉

  src="/images/stock/digital-garden.webp"

The inaugural release of Delta Components: 14 documented components and 9 ready-to-use blocks, built on top of the shadcn/ui framework and focused on the high-impact, production-ready interfaces that shadcn/ui doesn't cover out of the box.

### New Components

- **[Admonition](/docs/components/admonition)**: Semantic callout blocks for alerts and side content. **Seven types (note, warning, success, caution, danger, info, tip) with automatic icon selection and optional dismiss / collapsible variants.**
- **[Animated Tabs](/docs/components/tabs)**: Layered content sections with smooth CSS transitions. **Three variants (underlined, ghost, pill), entrance animations, and no external animation dependencies.**
- **[Cambio Image](/docs/components/cambio-image)**: Interactive, zoomable image component. **Physics-based motion, blur-up loading via IntersectionObserver, and gesture-driven dismissal.**
- **[Card Deck](/docs/components/card-deck)**: Swipeable, physics-based card stack. **Autoplay, touch gestures, and fully customizable motion presets for mobile-first interfaces.**
- **[Code Block](/docs/components/code-block)**: Versatile code display with syntax highlighting. **Parses Markdown fences, supports package-manager tabs, expandable viewport, and adaptive themes.**
- **[Input OTP](/docs/components/input-otp)**: Accessible one-time password input. **Copy-paste aware, with pill and separator variants for auth flows.**
- **[LLM Chat](/docs/components/chat)**: Complete AI chat interface. **Streaming responses, file uploads, model selection, and composable message/conversation/response primitives.**
- **[MapBox Pointer](/docs/components/mapbox-pointer)**: Interactive maps with custom React markers. **Built on Mapbox GL JS with theme-aware styling and declarative marker management.**
- **[Marquee](/docs/components/marquee)**: Animated, infinite-scrolling rail. **Perfect for logos, testimonials, or cards — with smooth continuous motion and direction control.**
- **[Navigation Menu](/docs/components/navigation-menu)**: Accessible link collection for site navigation. **Desktop and mobile-friendly link groups with keyboard support.**
- **[Product Card](/docs/components/product-card)**: Flexible compound component for commerce. **Multiple layout variants, distinct sizing, and interactive hover states.**
- **[QR Code](/docs/components/qrcode)**: Customizable QR code generator. **Theme-aware coloring, custom dot styling, and centered logo placement.**
- **[Scroll Fade Effect](/docs/components/scroll-fade-effect)**: Scroll container with edge fades. **Indicates available scroll content and supports vertical, horizontal, or bi-directional scroll.**
- **[X Card (Twitter)](/docs/components/x-card)**: Themed, rich embeds for X/Twitter posts. **Renders static, high-performance cards that integrate with your theme.**

### New Blocks

- **[AI Chat Sidebar](/blocks/ai-elements)**: Sidebar layout for AI chat applications with conversation history.
- **[Chatbot Window](/blocks/ai-elements)**: Resizable AI chatbot interface with a collapsible sidebar.
- **[Interactive Feature Showcase](/blocks/landing-page)**: Feature grid with interactive media cards for landing pages.
- **[Testimonials](/blocks/landing-page)**: Customer testimonials grid for social proof.
- **[MapBox Grid Block](/blocks/landing-page)**: Three-column grid showcasing global locations with Mapbox maps.
- **[Bottom Mobile Nav](/blocks/layout)**: Mobile-first layout with a bottom navigation bar, optimized for single-handed use.
- **[Admin Inset Layout](/blocks/dashboard)**: Admin dashboard layout with an inset sidebar, breadcrumbs, and command menu.
- **[SaaS Dashboard](/blocks/dashboard)**: SaaS dashboard with inset sidebar, navigation, and search command.
- **[Auth Form](/blocks/authentication)**: Authentication form supporting SSO, email, passkey, and SAML.
