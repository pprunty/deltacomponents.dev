## January 2026 - Beta Release

For the first round of components, I looked at what we build every day, the boring stuff we rebuild over and over, and made reusable abstractions you can actually use.

**These components work with every component library, Radix, Base UI, React Aria, you name it. Copy and paste to your projects.**

### Components

1. **[Admonition](/docs/components/admonition)**: Highlight side content and alerts with semantic colors and icons.
2. **[Animated Tabs](/docs/components/tabs)**: A set of layered sections of content—known as tab panels—that are displayed one at a time.
3. **[Cambio Image](/docs/components/cambio-image)**: An interactive image component with zoom functionality using the Cambio library.
4. **[Card Deck](/docs/components/card-deck)**: A swipeable card deck component with autoplay and infinite scroll options.
5. **[Code Block](/docs/components/code-block)**: A versatile code block component supporting package manager commands, syntax highlighting with themes, and markdown-wrapped code strings.
6. **[LLM Chat](/docs/components/chat)**: A complete chat component with streaming responses, file uploads, and model selection for building AI-powered applications.
7. **[Map Box Pointer](/docs/components/mapbox-pointer)**: An interactive map component with custom markers using Mapbox GL JS.
8. **[Marquee](/docs/components/marquee)**: An animated marquee component for showcasing any content with smooth scrolling and customizable styling. Perfect for logos, testimonials, cards, and more.
9. **[Navigation Menu](/docs/components/navigation-menu)**: A collection of links for navigating websites.
10. **[Product Card](/docs/components/product-card)**: A flexible compound component for displaying product information with images, badges, and metadata.
11. **[QR Code](/docs/components/qrcode)**: A customizable QR code component with styling options for Delta applications.
12. **[X Card (Twitter)](/docs/components/x-card)**: Embed X/Twitter posts with rich styling and theme integration. Perfect for showcasing tweets in documentation, blogs, and social content.

### Blocks

1. **[Testimonials](/blocks/landing-page#testimonials)**: Beautiful testimonial cards for showcasing customer reviews and feedback.
2. **[Video Card Grid](/blocks/landing-page#interactive-feature-showcase)**: An interactive feature showcase with video cards and smooth animations.
3. **[LLM Chat Window](/blocks/ai-elements#chatbot-window)**: A complete chat interface window for AI-powered conversations.
4. **[AI Chat Sidebar](/blocks/ai-elements#ai-chat-sidebar)**: A sidebar layout for AI chat applications with conversation history.
5. **[Perspective Carousel](/blocks/featured#perspective-carousel)**: A stunning 3D perspective carousel for showcasing content.
6. **[Bottom Mobile Nav Layout](/blocks/layout#bottom-mobile-nav)**: A mobile-first navigation layout with bottom navigation bar.

The experience in building these components was a joy. I focused meticulously on customization, variant design, and performance. Each component is built with accessibility in mind and works seamlessly with your existing design system.

Looking forward to the next release with more components and exciting features!
