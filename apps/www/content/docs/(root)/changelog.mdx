## January 2026 - Beta Release

  src="/images/stock/digital-garden.webp"

This release introduces 12 new components and 6 ready-to-use blocks, focused on high-impact, complex interfaces built on the shadcn/ui framework.

### New Components

- **[Admonition](/docs/components/admonition)**: Semantic callouts for alerts and side content. **Provides seven distinct types (e.g., warning, success, caution) with automatic icon selection.**
- **[Animated Tabs](/docs/components/tabs)**: Layered content sections with smooth transition animations. **Supports multiple variants, including underlined, ghost, and pill styles.**
- **[Cambio Image](/docs/components/cambio-image)**: Interactive, zoomable image component. **Features physics-based motion and blur-up loading via Intersection Observer.**
- **[Card Deck](/docs/components/card-deck)**: Swipeable card component. **Includes a 3D-effect stack and customizable swipe controls for mobile-friendly interfaces.**
- **[Code Block](/docs/components/code-block)**: Versatile code display block. **Features syntax highlighting, package manager tabs, and an expandable viewport for long code snippets.**
- **[LLM Chat](/docs/components/chat)**: Complete AI chat interface. **Supports streaming responses, state management, file uploads, and model selection for a full-featured application.**
- **[Map Box Pointer](/docs/components/mapbox-pointer)**: Interactive maps. **Uses Mapbox GL JS to render custom React markers and manage map state.**
- **[Marquee](/docs/components/marquee)**: Animated, infinite scrolling rail. **Perfect for showcasing logos, testimonials, or feature cards with smooth, continuous motion.**
- **[Navigation Menu](/docs/components/navigation-menu)**: An accessible collection of links. **Offers desktop and mobile-friendly link groups for site navigation.**
- **[Product Card](/docs/components/product-card)**: A flexible compound component. **Designed for e-commerce product listings or showcasing, easily combining images, badges, and metadata.**
- **[QR Code](/docs/components/qrcode)**: Customizable QR code generator. **Includes styling options for colors and supports centered logo placement within the code.**
- **[X Card (Twitter)](/docs/components/x-card)**: Themed, rich embeds for X/ (formerly Twitter) posts. **Renders static, high-performance cards that integrate seamlessly with your theme.**

### New Blocks

- **[Testimonials](/blocks/landing-page#testimonials)**: Feature showcase with testimonial cards. **A dedicated section for displaying customer reviews and social proof.**
- **[Video Card Grid](/blocks/landing-page#interactive-feature-showcase)**: Interactive feature grid with video cards. **Highlights product features using dynamic media and responsive layouts.**
- **[LLM Chat Window](/blocks/ai-elements#chatbot-window)**: Complete AI chat interface window layout. **A full-screen, ready-to-use window template for AI applications.**
- **[AI Chat Sidebar](/blocks/ai-elements#ai-chat-sidebar)**: Sidebar layout for AI chat applications. **Includes conversation history navigation and primary chat interface.**
- **[Perspective Carousel](/blocks/featured#perspective-carousel)**: A stunning 3D perspective carousel. **Showcases key content with an immersive, three-dimensional depth effect.**
- **[Bottom Mobile Nav Layout](/blocks/layout#bottom-mobile-nav)**: Mobile-first layout with bottom navigation bar. **Optimized for single-handed use on mobile devices.**

---

The experience building these components was a joy. I focused meticulously on customization, variant design, and performance. Each component is built with accessibility in mind and works seamlessly with your existing design system.

Looking forward to the next release with more components and exciting features!
