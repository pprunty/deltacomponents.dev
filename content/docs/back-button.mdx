# BackButton

A smart back button component that intelligently handles navigation history with a fallback to root.

## Features

- **Smart Navigation**: Automatically detects browser history and navigates back when possible
- **Fallback Handling**: Falls back to a specified route (defaults to '/') when no history is available
- **Customizable**: Supports text, variants, and icon customization
- **Accessible**: Built with proper ARIA attributes and keyboard navigation

## Smart Navigation Handling

The BackButton component intelligently handles navigation:

1. When clicked, it first checks if there's browser history available (`window.history.length > 2`).
2. If history exists, it uses `router.back()` to navigate to the previous page.
3. If no history is available (e.g., user landed directly on the page), it navigates to the `fallbackRoute`.
4. The default fallback route is the root path (`'/'`), but this can be customized via the `fallbackRoute` prop.

```tsx
import { BackButton } from "@/delta/components/back-button"

export default function BackButtonExample() {
  return (
    // Will go back in history if available, otherwise to /dashboard
    <BackButton 
      text="Back to Dashboard"
      fallbackRoute="/dashboard"
      variant="ghost"
      size="md"
    />
  )
}
```
