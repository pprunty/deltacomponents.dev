# TextEditor

A rich text editor component built with TipTap, featuring text formatting, lists, links, and customizable styling.

## Basic Usage

<ComponentTabs name="text-editor-basic" />

## Installation

```bash
npx shadcn@latest add "text-editor"
```

## Features

- **Rich Text Editing**: Full support for rich text formatting including:
  - **Bold** text (Ctrl/Cmd + B)
  - **Italic** text (Ctrl/Cmd + I)
  - **Lists** (bullet and ordered)
  - **Links** with custom styling
- **Customizable**: Adjust styling, placeholder text, and toolbar visibility
- **Accessibility**: Built with accessibility in mind
- **Responsive**: Works well on all screen sizes
- **Dark Mode Support**: Automatically adapts to your theme

## Usage Examples

### Basic Text Editor

```tsx
import { TextEditor } from "@/components/ui/text-editor"

export default function TextEditorDemo() {
  return <TextEditor />
}
```

### With Initial Content

```tsx
import { TextEditor } from "@/components/ui/text-editor"

export default function TextEditorDemo() {
  return (
    <TextEditor 
      initialContent={`
        <h2>Welcome to Delta Components</h2>
        <p>A collection of modern, accessible, and customizable React components built on top of shadcn/ui.</p>
        
        <h3>Key Features</h3>
        <ul>
          <li><strong>Modern Design</strong>: Clean and contemporary UI components</li>
          <li><em>Accessibility First</em>: Built with WCAG guidelines in mind</li>
          <li><strong><em>Customizable</em></strong>: Easy to style and extend</li>
        </ul>
        
        <p>Learn more at <a href="https://deltacomponents.dev">deltacomponents.dev</a></p>
      `}
      onChange={(html) => console.log(html)}
    />
  )
}
```

### Custom Controls

```tsx
import { TextEditor } from "@/components/ui/text-editor"

export default function TextEditorDemo() {
  return (
    <TextEditor 
      controls={{
        bold: true,
        italic: true,
        bulletList: false,
        orderedList: false,
        link: true
      }}
    />
  )
}
```

## Keyboard Shortcuts

The text editor supports the following keyboard shortcuts:

- **Bold**: `Ctrl/Cmd + B`
- **Italic**: `Ctrl/Cmd + I`
- **Bullet List**: `Ctrl/Cmd + Shift + 8`
- **Ordered List**: `Ctrl/Cmd + Shift + 7`
- **Link**: `Ctrl/Cmd + K`

## Accessibility

The TextEditor component follows accessibility best practices:

- Proper ARIA attributes for the editor and toolbar
- Keyboard navigation support
- Focus management
- Screen reader compatibility
- High contrast support
