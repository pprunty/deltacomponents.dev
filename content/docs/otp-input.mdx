## Usage

```tsx
import { OTPInput } from "@/components/ui/otp-input"

export default function OTPInputDemo() {
  return (
    <OTPInput 
      label="Verification Code"
      name="otp"
      length={6}
      required
    />
  )
}
```

## Validation

The component integrates with Zod for validation:

```tsx
import { OTPInput } from "@/components/ui/otp-input"
import { z } from "zod"

const otpSchema = z.string().length(6).regex(/^\d+$/)

export default function OTPInputDemo() {
  return (
    <OTPInput 
      label="Verification Code"
      name="otp"
      schema={otpSchema}
      onValidate={(isValid, value) => {
        console.log('Validation result:', isValid, value)
      }}
    />
  )
}
```

## Accessibility

The OTP Input component follows WAI-ARIA guidelines and includes proper ARIA attributes for accessibility. It supports keyboard navigation, screen readers, and provides clear error messaging.
