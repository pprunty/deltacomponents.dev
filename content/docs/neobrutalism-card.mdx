## Subcomponents

The NeobrutalismCard component includes several subcomponents for structured content:

- `CardHeader`: Container for the card's header content
- `CardTitle`: The main title of the card
- `CardDescription`: A subtitle or description for the card
- `CardContent`: The main content area of the card
- `CardFooter`: Container for footer content

## Usage

```tsx
import {
  Card,
  CardHeader,
  CardTitle,
  CardDescription,
  CardContent,
  CardFooter,
} from "@/components/ui/neobrutalism-card"

export default function NeobrutalismCardDemo() {
  return (
    <Card color="blue" hover={true}>
      <CardHeader>
        <CardTitle>Card Title</CardTitle>
        <CardDescription>Card Description</CardDescription>
      </CardHeader>
      <CardContent>
        <p>Your card content goes here.</p>
      </CardContent>
      <CardFooter>
        <p>Footer content</p>
      </CardFooter>
    </Card>
  )
}
```

## Customization

You can customize the card's appearance using the available props:

```tsx
<Card 
  color="pink" 
  hover={true} 
  borderWidth="thick"
  className="max-w-md"
>
  {/* Card content */}
</Card>
```
