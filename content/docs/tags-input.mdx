# TagsInput

A versatile tags input component that allows users to add and remove tags with keyboard support. Features include validation, error handling, and various styling options.

## Basic Usage

<ComponentTabs name="tags-input-basic" />

## Installation

```bash
npx shadcn@latest add "tags-input"
```

## Features

- **Keyboard Support**: Add tags using Enter, Space, or Comma keys
- **Tag Removal**: Remove tags by clicking the X button or using Backspace
- **Validation Support**: Built-in error handling and validation with Zod
- **Customizable**: Supports custom styling and variants
- **Accessibility**: Fully accessible with proper ARIA attributes
- **Flexible Input**: Supports both controlled and uncontrolled usage
- **Form Integration**: Works seamlessly with Smart Form component
- **Loading State**: Supports pending/loading state
- **Label Variants**: Supports default and muted label styles
- **Colored Border**: Optional colored border for pill variant

## Examples

### Basic Usage

```tsx
import { TagsInput } from "@/components/ui/tags-input"

export default function Example() {
  return (
    <TagsInput
      label="Tags"
      name="tags"
      placeholder="Add tags..."
    />
  )
}
```

### With Validation

```tsx
import { TagsInput } from "@/components/ui/tags-input"
import { z } from "zod"

const schema = z.array(z.string())
  .min(1, "At least one tag is required")
  .max(5, "Maximum 5 tags allowed")

export default function ValidationExample() {
  return (
    <TagsInput
      label="Tags"
      name="tags"
      schema={schema}
      hint="Add up to 5 tags"
    />
  )
}
```

### Pill Variant with Colored Border

```tsx
import { TagsInput } from "@/components/ui/tags-input"

export default function Example() {
  return (
    <TagsInput
      label="Tags"
      name="tags"
      variant="pill"
      coloredBorder={true}
      placeholder="Add tags..."
    />
  )
}
```

### With Loading State

```tsx
import { TagsInput } from "@/components/ui/tags-input"
import { useState } from "react"

export default function LoadingExample() {
  const [pending, setPending] = useState(false)
  
  return (
    <TagsInput
      label="Tags"
      name="tags"
      pending={pending}
      placeholder="Add tags..."
    />
  )
}
```

### With Smart Form

```tsx
import { SmartForm } from "@/components/ui/smart-form"
import { z } from "zod"

const schema = z.object({
  tags: z.array(z.string())
    .min(1, "At least one tag is required")
    .max(5, "Maximum 5 tags allowed")
})

const fields = [
  {
    type: "tags",
    name: "tags",
    label: "Tags",
    description: "Add tags by pressing space, comma, or enter",
    hint: "Press backspace to remove the last tag",
    variant: "pill",
    triggerKey: "Enter",
    required: true
  }
]

export default function SmartFormExample() {
  return (
    <SmartForm
      fields={fields}
      schema={schema}
      onSubmit={(data) => console.log(data)}
    />
  )
}
```

## Accessibility

The TagsInput component is built with accessibility in mind:

- Proper labeling and ARIA attributes
- Support for required field indicators
- Error messages are properly associated with the input
- Keyboard navigation support
- Clear focus states for keyboard users
- Proper button roles for tag removal
- Support for screen readers
- Proper color contrast ratios
- Disabled state handling
- Loading state indicators
