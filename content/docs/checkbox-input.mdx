## Examples

### Basic Example

```tsx
import { CheckboxInput } from "@/components/ui/checkbox-input"

export default function Example() {
  return (
    <CheckboxInput
      label="Accept terms and conditions"
      name="terms"
      description="By accepting, you agree to our Terms of Service and Privacy Policy"
      required
    />
  )
}
```

### With Hint and Error

```tsx
import { CheckboxInput } from "@/components/ui/checkbox-input"

export default function Example() {
  return (
    <CheckboxInput
      label="Subscribe to newsletter"
      name="newsletter"
      hint="We'll send you weekly updates and offers"
      error="You must accept to continue"
    />
  )
}
```

### Controlled Component

```tsx
import { useState } from "react"
import { CheckboxInput } from "@/components/ui/checkbox-input"

export default function ControlledExample() {
  const [isChecked, setIsChecked] = useState(false)
  
  return (
    <CheckboxInput
      label="I agree to receive marketing emails"
      name="marketing"
      checked={isChecked}
      onCheckedChange={setIsChecked}
    />
  )
}
```

## Accessibility

The CheckboxInput component is built with accessibility in mind:

- Proper labeling and ARIA attributes
- Support for required field indicators
- Error messages are properly associated with the checkbox
- Keyboard navigation support
- Clear focus states for keyboard users
