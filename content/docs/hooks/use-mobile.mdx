## Overview

The `useMobile` hook provides a simple way to detect if the current viewport width is below the mobile breakpoint (640px). It automatically updates on window resize and is useful for conditional rendering based on screen size.

## API Reference

### Return Value

| Type    | Description                                                        |
| ------- | ------------------------------------------------------------------ |
| boolean | `true` if the viewport width is less than 640px, `false` otherwise |

## Features

- **Reactive**: Automatically updates on window resize
- **Performance**: Uses efficient event listeners with proper cleanup
- **SSR-safe**: Handles server-side rendering gracefully
- **Tailwind-aligned**: Uses the same breakpoint as Tailwind's `sm` (640px)
