# FloatingButton

A versatile floating button component with customizable position, tooltip support, and responsive behavior. Perfect for primary actions like creating new items or navigating back to top.

## Basic Usage

<ComponentTabs name="floating-button-basic" />

## Installation

```bash
npx shadcn@latest add "floating-button"
```

## Examples

### Basic Floating Button with Icon Only

```tsx
import { Plus } from "lucide-react"
import { FloatingButton } from "@/components/ui/floating-button"

export default function BasicExample() {
  return (
    <FloatingButton.FloatingButton
      icon={Plus}
      shape="circle"
      tooltip="Add New Item"
    />
  )
}
```

### Floating Button with Text and Icon

```tsx
import { ArrowUp } from "lucide-react"
import { FloatingButton } from "@/components/ui/floating-button"

export default function TextExample() {
  return (
    <FloatingButton.FloatingButton
      icon={ArrowUp}
      text="Back to Top"
      placement="bottom-center"
    />
  )
}
```

### Different Variants and Sizes

```tsx
import { Heart, Send, Trash } from "lucide-react"
import { FloatingButton } from "@/components/ui/floating-button"

export default function VariantsExample() {
  return (
    <div className="space-y-4">
      <FloatingButton.FloatingButton
        icon={Heart}
        variant="secondary"
        size="sm"
        position="static"
      />
      
      <FloatingButton.FloatingButton
        icon={Send}
        variant="outline"
        position="static"
      />
      
      <FloatingButton.FloatingButton
        icon={Trash}
        variant="destructive"
        size="lg"
        position="static"
      />
    </div>
  )
}
```

### Responsive Positioning

```tsx
import { Plus } from "lucide-react"
import { FloatingButton } from "@/components/ui/floating-button"

export default function ResponsiveExample() {
  return (
    <FloatingButton.FloatingButton
      icon={Plus}
      text="Add Item"
      mobilePosition="bottom-center"
      desktopPosition="bottom-right"
      mobileOffset={8}
      desktopOffset={16}
    />
  )
}
```

## Accessibility

The FloatingButton component is designed with accessibility in mind:

- Includes proper ARIA attributes
- Supports keyboard navigation
- Tooltips improve discoverability for icon-only buttons
- Text alternatives can be provided for screen readers

When using icon-only FloatingButtons, it's recommended to include a tooltip or aria-label to describe the action for assistive technology users.
