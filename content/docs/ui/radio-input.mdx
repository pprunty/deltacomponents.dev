## Examples

### Basic Example

```tsx
import { RadioInput } from "@/components/ui/radio-input"

export default function Example() {
  const options = [
    { value: "option1", label: "Option 1" },
    { value: "option2", label: "Option 2" },
    { value: "option3", label: "Option 3" }
  ]

  return (
    <RadioInput
      label="Select an option"
      name="options"
      options={options}
      defaultValue="option1"
    />
  )
}
```

### Horizontal Layout

```tsx
import { RadioInput } from "@/components/ui/radio-input"

export default function HorizontalExample() {
  const options = [
    { value: "yes", label: "Yes" },
    { value: "no", label: "No" }
  ]

  return (
    <RadioInput
      label="Proceed with action?"
      name="confirm"
      options={options}
      orientation="horizontal"
    />
  )
}
```

### With Descriptions and Controlled State

```tsx
import { useState } from "react"
import { RadioInput } from "@/components/ui/radio-input"

export default function ControlledExample() {
  const [selectedOption, setSelectedOption] = useState("basic")
  
  const options = [
    { 
      value: "basic", 
      label: "Basic Plan", 
      description: "$10/month - Access to basic features" 
    },
    { 
      value: "pro", 
      label: "Pro Plan", 
      description: "$25/month - Access to all features" 
    },
    { 
      value: "enterprise", 
      label: "Enterprise Plan", 
      description: "$100/month - Priority support and custom features" 
    }
  ]
  
  return (
    <RadioInput
      label="Select Subscription Plan"
      name="plan"
      options={options}
      value={selectedOption}
      onValueChange={setSelectedOption}
    />
  )
}
```

## Accessibility

The RadioInput component is built with accessibility in mind:

- Proper labeling and ARIA attributes
- Support for required field indicators
- Error messages are properly associated with the radio group
- Keyboard navigation support
- Clear focus states for keyboard users
