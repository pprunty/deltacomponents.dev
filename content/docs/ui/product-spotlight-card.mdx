## Usage

```tsx
import ProductSpotlightCard from "@/components/ui/product-spotlight-card"

export default function ProductSpotlightCardDemo() {
  return (
    <div className="space-y-8">
      {/* Interactive card with href */}
      <ProductSpotlightCard
        imageSrc="/stock/switch.png"
        imageAlt="Nintendo Switch"
        title="Nintendo Switch"
        description="The Nintendo Switch is a hybrid video game console developed by Nintendo."
        href="/products/switch"
      />

      {/* Static card without href */}
      <ProductSpotlightCard
        imageSrc="/stock/switch.png"
        imageAlt="Nintendo Switch"
        title="Nintendo Switch"
        description="The Nintendo Switch is a hybrid video game console developed by Nintendo."
      />
    </div>
  )
}
```

## Features

- **Responsive Design**: Automatically adjusts to different screen sizes
- **Hover Effects**: Smooth transitions and visual feedback on hover (only when href is provided)
- **Accessibility**: Proper ARIA attributes and keyboard navigation
- **Image Optimization**: Built-in Next.js Image component for optimal loading
- **Customizable**: Supports custom styling through className prop
- **Flexible Layout**: Works with or without title and description
- **Interactive States**: Automatically adapts to be interactive or static based on href prop

## Image Preparation

For best results, use product images with transparent backgrounds. On macOS, you can quickly remove backgrounds from images:

1. Right-click on the image file
2. Select "Quick Actions" > "Remove Background"
3. The system will automatically process the image and create a new version with a transparent background
4. Save the processed image and use it in your ProductSpotlightCard

This works best with high-contrast product images and will help the card's gradient background show through more effectively.

## Accessibility

The ProductSpotlightCard component follows accessibility best practices:
- Proper ARIA attributes for screen readers
- Keyboard navigation support
- Semantic HTML structure
- Focus management
- High contrast ratios for text
