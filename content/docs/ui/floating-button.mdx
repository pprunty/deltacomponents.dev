# Float

A versatile positioning component that can float any content in fixed or absolute positions with responsive behavior. Perfect for floating action buttons, notifications, or any other content that needs to be positioned relative to the viewport or a container.

## Examples

### Basic Floating Button

```tsx
import { Plus } from "lucide-react"
import Float from "@/components/ui/float"
import { Button } from "@/components/ui/button"

export default function BasicExample() {
  return (
    <Float
      position="absolute"
      placement="bottom-right"
      offset={16}
    >
      <Button
        variant="primary"
        size="sm"
        onClick={() => alert("Button clicked!")}
      >
        <Plus size={20} />
      </Button>
    </Float>
  )
}
```

### Floating Notification

```tsx
import Float from "@/components/ui/float"

export default function NotificationExample() {
  return (
    <Float
      placement="top-right"
      offset={16}
    >
      <div className="bg-primary text-primary-foreground px-4 py-2 rounded-md shadow-lg">
        New message received!
      </div>
    </Float>
  )
}
```

### Responsive Positioning

```tsx
import Float from "@/components/ui/float"
import { Button } from "@/components/ui/button"

export default function ResponsiveExample() {
  return (
    <Float
      mobilePosition="bottom-center"
      desktopPosition="bottom-right"
      mobileOffset={8}
      desktopOffset={16}
    >
      <Button variant="primary">
        Contact Us
      </Button>
    </Float>
  )
}
```

### Opacity Changes on Scroll

```tsx
import Float from "@/components/ui/float"
import { Button } from "@/components/ui/button"

export default function OpaqueOnScrollExample() {
  return (
    <Float
      placement="bottom-right"
      opaqueOnScroll={true}
    >
      <Button variant="primary">
        Back to Top
      </Button>
    </Float>
  )
}
```

## Accessibility

The Float component is designed with accessibility in mind:

- Maintains proper focus management
- Supports keyboard navigation
- Preserves semantic structure of its children
- Works with screen readers

When using the Float component, ensure that the content you're floating is itself accessible. For example, when floating buttons, make sure they have proper ARIA labels and keyboard support.
