# Tweet

A component for embedding and displaying tweets with server-side rendering, dark mode support, and responsive design.

## Installation

### CLI

```shell
npx shadcn@latest add "https://deltacomponents.dev/r/tweet.json"
```

## Basic Usage

The Tweet component can display tweets in both default and small sizes.

<ComponentTabs name="tweet-basic" />

## Styles and Customization

The Tweet component supports various styling options and sizes.

<ComponentTabs name="tweet-styles" />

## Usage

```tsx
import { Tweet } from "@/components/ui/tweet"

export default function TweetDemo() {
  return (
    <Tweet id="1726130933966856722" />
  )
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `id` | `string` | - | The ID of the tweet to display |
| `small` | `boolean` | `false` | Whether to display the tweet in a smaller size |
| `className` | `string` | - | Additional CSS classes to apply to the tweet container |

## Features

- Server-side rendering for better performance
- Dark mode support
- Responsive design
- Automatic retry mechanism for failed requests
- Loading states and error handling
- Custom styling support

## Accessibility

The Tweet component preserves Twitter's built-in accessibility features and maintains proper contrast in both light and dark modes.
