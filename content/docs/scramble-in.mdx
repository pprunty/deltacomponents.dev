## Intersection Observer Demo

The ScrambleIn component can be configured to trigger the animation when the text comes into view using the Intersection Observer API. This is perfect for scroll-based animations and page transitions.

<ComponentTabs name="scramble-in-observer" />

## Features

- Customizable animation speed
- Configurable scramble effect
- Intersection Observer support
- Custom character sets
- Callback support
- Accessibility features

## Usage

```tsx
import ScrambleIn from "@/delta/components/scramble-in"

export default function ScrambleInDemo() {
  return (
    <ScrambleIn 
      text="Welcome to Delta Components"
      className="text-3xl font-bold"
      scrambledClassName="text-3xl font-bold"
      scrambleSpeed={50}
      scrambledLetterCount={3}
    />
  )
}
```

## Custom Styling

You can customize the text styles using Tailwind CSS classes:

```tsx
<ScrambleIn 
  text="Custom Style"
  className="text-4xl font-black text-blue-500"
  scrambledClassName="text-4xl font-black text-blue-500"
/>
```

## Accessibility

The ScrambleIn component includes proper ARIA attributes and maintains accessibility by:
- Providing screen reader text
- Preserving semantic meaning
- Supporting keyboard navigation
- Maintaining proper contrast
