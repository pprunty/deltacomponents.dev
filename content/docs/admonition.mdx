## Usage

```tsx
import { Admonition } from "@/delta/components/admonition"

export default function AdmonitionDemo() {
  return (
    <Admonition type="info" title="Did you know?">
      This is an informational message.
    </Admonition>
  )
}
```

## Custom Styling

You can customize the admonition styles using Tailwind CSS classes:

```tsx
<Admonition 
  type="info" 
  className="my-8 rounded-lg"
>
  Custom styled admonition
</Admonition>
```

## Accessibility

The admonition component follows WAI-ARIA guidelines and includes proper ARIA attributes for accessibility. Each admonition type has a distinct visual style and semantic meaning to ensure clear communication of its purpose.
