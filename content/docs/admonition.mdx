# Admonition

A flexible component for displaying notices, warnings, tips, and other important information with customizable styles and icons.

## Installation

```bash
npx shadcn@latest add "admonition"
```

## Usage

```tsx
import { Admonition } from "@/components/ui/admonition"
import { AlertCircle, Info, Lightbulb, AlertTriangle } from "lucide-react"

export default function AdmonitionDemo() {
  return (
    <div className="space-y-4">
      <Admonition type="note" title="Important Note">
        This is an important message that users should pay attention to.
      </Admonition>

      <Admonition type="warning" title="Custom Icon" icon={<AlertCircle className="h-5 w-5" />}>
        This admonition uses a custom icon.
      </Admonition>
    </div>
  )
}
```

## Customization

You can customize the admonition's appearance using Tailwind CSS classes:

```tsx
<Admonition 
  type="note" 
  className="border-l-4 border-blue-500 bg-blue-50 dark:bg-blue-900/30"
>
  Custom styled note
</Admonition>
```

## Accessibility

The admonition component follows WAI-ARIA guidelines and includes proper ARIA attributes for accessibility. Each type has appropriate role and aria-label attributes to ensure screen readers can properly announce the content.
