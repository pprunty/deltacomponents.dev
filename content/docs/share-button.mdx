# ShareButton

A versatile share button component that leverages the Web Share API for mobile devices and falls back to clipboard copy functionality for desktop browsers.

## Basic Usage

<ComponentTabs name="share-button-basic" />

## Installation

```bash
npx shadcn@latest add "share-button"
```

## Features

- **Native Share API Support**: Uses the Web Share API on compatible devices for a native sharing experience
- **Copy to Clipboard Fallback**: Falls back to copying the URL to clipboard on desktop or unsupported browsers
- **Visual Feedback**: Shows a confirmation message after successful copy operations
- **Customizable**: Supports custom messages, URLs, styles, and icons
- **Accessibility**: Fully accessible with proper ARIA attributes
- **Flexible Desktop Behavior**: Option to use native share API on desktop or default clipboard copy

## Usage Examples

### Basic Share Button

```tsx
import { ShareButton } from "@/components/ui/share-button";

export default function ShareExample() {
  return <ShareButton />;
}
```

### Custom Content and Styling

```tsx
import { ShareButton } from "@/components/ui/share-button";
import { Share } from "@phosphor-icons/react";

export default function ShareExample() {
  return (
    <ShareButton 
      tooltip="Share this content"
      className="hover:bg-blue-100 dark:hover:bg-blue-900/30"
    >
      <div className="flex items-center gap-2">
        <Share weight="bold" size={24} />
        <span>Share</span>
      </div>
    </ShareButton>
  );
}
```

### Custom Icon and Styling (Legacy)

```tsx
import { ShareButton } from "@/components/ui/share-button";
import { Share } from "@phosphor-icons/react";

export default function ShareExample() {
  return (
    <ShareButton 
      icon={<Share weight="bold" size={24} />}
      variant="outline"
      size="lg"
      className="hover:bg-blue-100 dark:hover:bg-blue-900/30"
    />
  );
}
```

### Custom URL and Message

```tsx
import { ShareButton } from "@/components/ui/share-button";

export default function ShareExample() {
  return (
    <ShareButton 
      url="https://deltacomponents.dev"
      message="Check out Delta Components, a modern UI component library: "
      title="Delta Components"
    >
      <div className="flex items-center gap-2">
        <span>Share Delta Components</span>
      </div>
    </ShareButton>
  );
}
```

### Native Share API on Desktop

```tsx
import { ShareButton } from "@/components/ui/share-button";

export default function ShareExample() {
  return (
    <ShareButton 
      nativeShareOnDesktop={true}
      tooltip="Share using native dialog"
    >
      <div className="flex items-center gap-2">
        <span>Share with Native Dialog</span>
      </div>
    </ShareButton>
  );
}
```

## Browser Compatibility

The Web Share API is supported in:
- Chrome for Android (61+)
- Safari for iOS (12.2+)
- Safari on macOS (Monterey+)
- Chrome on desktop (89+)
- Edge (79+)
- Opera (76+)

On unsupported browsers or when `nativeShareOnDesktop` is `false` on desktop, the component falls back to copying the URL to clipboard and displaying a notification.

## Accessibility

The ShareButton component follows accessibility best practices:

- Includes proper `aria-label` attributes
- Uses semantic HTML for the button element
- Provides visual feedback on successful actions
- Maintains focus management best practices
- Uses appropriate color contrast ratios
