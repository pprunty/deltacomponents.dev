A modern, playful variant inspired by the neobrutalism design trend. Features bold shadows, thick borders, and interactive animations.

Buttons can include icons alongside text, be icon-only, or use the minimal ghost variant for subtle interactions.

Buttons can have an extended click area that reaches beyond their visible boundaries, making them easier to click, especially on touch devices.

## Usage

```tsx
import { Button } from "@/components/ui/button"

export default function ButtonDemo() {
  return (
    <Button variant="primary" title="Click me" />
  )
}
```

## Custom Styling

You can customize the button styles using Tailwind CSS classes:

```tsx
<Button className="bg-custom hover:bg-custom-dark">
  Custom Style
</Button>
```

## Accessibility

The button component follows WAI-ARIA guidelines and includes proper ARIA attributes for accessibility. Icon-only buttons should include an aria-label for screen readers.
