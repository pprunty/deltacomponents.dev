## Usage

```tsx
import { Button } from "@/components/ui/button"

export default function ButtonDemo() {
  return (
    <div className="space-y-4">
      {/* Using the title prop (preferred) */}
      <Button variant="primary" title="Click me" />
      
      {/* Using children */}
      <Button variant="secondary">Click me too</Button>
    </div>
  )
}
```

## Custom Styling

You can customize the button styles using Tailwind CSS classes:

```tsx
<Button className="bg-custom hover:bg-custom-dark">
  Custom Style
</Button>
```

## Accessibility

The button component follows WAI-ARIA guidelines and includes proper ARIA attributes for accessibility. Icon-only buttons should include an aria-label for screen readers.
