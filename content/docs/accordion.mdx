# Accordion

A versatile accordion component that extends shadcn/ui with additional features like chevron positioning, variants, and sizes. Perfect for organizing and displaying collapsible content sections.

## Basic Usage

<ComponentTabs name="accordion-basic" />

<InstallationInstructions componentName="accordion" />

## Features

- **Multiple Variants**: Choose from default, bordered, and ghost styles
- **Customizable Sizes**: Small, medium, and large text sizes
- **Flexible Chevron**: Position chevrons on the left or right
- **Chevron Variants**: Choose between down and right chevron styles
- **Accessibility**: Fully accessible with proper ARIA attributes
- **Animations**: Smooth open/close animations
- **Type Support**: Single or multiple item expansion

## Usage Examples

### Basic Accordion

```tsx
import { Accordion, AccordionItem, AccordionTrigger, AccordionContent } from "@/delta/components/accordion"

export default function AccordionDemo() {
  return (
    <Accordion type="single" collapsible>
      <AccordionItem value="item-1">
        <AccordionTrigger>Is it accessible?</AccordionTrigger>
        <AccordionContent>
          Yes. It adheres to the WAI-ARIA design pattern.
        </AccordionContent>
      </AccordionItem>
      <AccordionItem value="item-2">
        <AccordionTrigger>Is it styled?</AccordionTrigger>
        <AccordionContent>
          Yes. It comes with default styles that match the other components.
        </AccordionContent>
      </AccordionItem>
    </Accordion>
  )
}
```

### Bordered Variant with Left Chevron

```tsx
import { Accordion, AccordionItem, AccordionTrigger, AccordionContent } from "@/delta/components/accordion"

export default function AccordionDemo() {
  return (
    <Accordion variant="bordered" size="lg">
      <AccordionItem value="item-1">
        <AccordionTrigger chevronPosition="left">
          What is Delta Components?
        </AccordionTrigger>
        <AccordionContent>
          Delta Components is a collection of reusable React components built on top of shadcn/ui.
        </AccordionContent>
      </AccordionItem>
    </Accordion>
  )
}
```

### Multiple Items with Right Chevron

```tsx
import { Accordion, AccordionItem, AccordionTrigger, AccordionContent } from "@/delta/components/accordion"

export default function AccordionDemo() {
  return (
    <Accordion type="multiple">
      <AccordionItem value="item-1">
        <AccordionTrigger chevronVariant="right">
          How do I install?
        </AccordionTrigger>
        <AccordionContent>
          You can install using the CLI command: npx shadcn@latest add "accordion"
        </AccordionContent>
      </AccordionItem>
      <AccordionItem value="item-2">
        <AccordionTrigger chevronVariant="right">
          Can I customize it?
        </AccordionTrigger>
        <AccordionContent>
          Yes, you can customize the component using the provided props and Tailwind CSS.
        </AccordionContent>
      </AccordionItem>
    </Accordion>
  )
}
```

## Accessibility

The Accordion component follows WAI-ARIA guidelines and includes:

- Proper ARIA attributes for expanded/collapsed states
- Keyboard navigation support
- Focus management
- Screen reader compatibility
- Semantic HTML structure

## Animations

The component includes smooth animations for opening and closing accordion items. These animations are implemented using CSS transitions and can be customized using Tailwind CSS classes.
