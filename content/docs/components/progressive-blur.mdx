A progressive blur component that creates smooth fade and blur effects at the top or bottom of containers. Perfect for creating scroll fade effects, content overlays, and modern UI transitions. Uses multiple blur layers and automatic theme integration.

<ComponentPreview name="progressive-blur-basic-demo" />

## Overview

The ProgressiveBlur component applies a progressive blur effect using backdrop filtering combined with CSS gradients and masking. It automatically uses your theme's background color for seamless integration with light and dark modes. The blur effect can be toggled with the `blurEnabled` prop.

## Usage

### Basic Usage

```tsx
import ProgressiveBlur from "@/registry/components/progressive-blur"

export function Component() {
  return (
    <div className="relative h-96 bg-background overflow-hidden">
      <ProgressiveBlur position="top" height="150px" blurAmount="4px" />
      {/* Your content here */}
    </div>
  )
}
```

### Disabled Blur (Mask Only)

```tsx
import ProgressiveBlur from "@/registry/components/progressive-blur"

export function Component() {
  return (
    <div className="relative h-96 bg-background overflow-hidden">
      <ProgressiveBlur position="top" height="150px" blurEnabled={false} />
      {/* Your content here */}
    </div>
  )
}
```

## API Reference

| Name              | Type                | Default                    | Description                                                              |
| ----------------- | ------------------- | -------------------------- | ------------------------------------------------------------------------ |
| `className`       | `string`            | `""`                       | Additional CSS classes to apply to the blur overlay                      |
| `backgroundColor` | `string`            | `"hsl(var(--background))"` | Background color for the gradient. Defaults to theme background          |
| `position`        | `"top" \| "bottom"` | `"top"`                    | Whether to apply the blur effect at the top or bottom                    |
| `height`          | `string`            | `"150px"`                  | Height of the blur effect area                                           |
| `blurAmount`      | `string`            | `"4px"`                    | Intensity of the backdrop blur effect                                    |
| `blurEnabled`     | `boolean`           | `true`                     | Whether to apply the blur effect. When false, shows only the mask effect |

### Theme Integration

The component automatically uses your shadcn theme's background color via `hsl(var(--background))`. This ensures:

- **Automatic theme switching**: Works seamlessly with light and dark modes
- **Consistent design**: Matches your theme's color palette perfectly
- **Custom backgrounds**: Override with any CSS color value when needed

### Examples

```tsx
// Basic blur effect (recommended)
<ProgressiveBlur position="top" height="150px" />

// Custom height and blur amount
<ProgressiveBlur position="bottom" height="200px" blurAmount="8px" />

// Mask-only effect (no blur)
<ProgressiveBlur position="top" blurEnabled={false} />

// Custom background color
<ProgressiveBlur position="top" backgroundColor="hsl(var(--card))" />

// Multiple positions
<>
  <ProgressiveBlur position="top" />
  <ProgressiveBlur position="bottom" />
</>
```
